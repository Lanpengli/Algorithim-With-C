# Algorithm-With-C
#### 使用环境

* Ubuntu18.04 LTS
* g++ 7.3.0
* CMake 3.10.2
* Vim 8.0.1453
* C++11

#### 常用算法总结

* 选择排序、快速排序、希尔排序和堆排序是不稳定的排序算法
* 冒泡排序、插入排序、归并排序和基数排序是稳定的排序算法

![Sort](Sort.png)

#### 如何使用代码

* 新建build文件夹 `mkdir build`
* 进入build文件夹 `cd build`
* cmake配置 `cmake ..`
* make 编译 `make`
* 执行代码 `./Algorithm`
